University College of Northern Denmark

This website/prototype was created in the CMS DynamicWeb as part of my internship at UCN on the 8th semester.

The prototype displays my idea for a redesign of the website structure/design for the educations overview, start of term, and admission pages. The design was based on empirical data collected through several usability tests on the current website, which reveraled several findability, navigational, and labeling issues. The new prototype was tested for these issues, and the test showed that they were no longer present.

Since the prototype was created in the CMS system at UCN, it is not accessible to the public.
